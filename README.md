# GAN_Detector
---
## 설치 방법

### 1. 원하는 버전의 GAN_Detector 릴리즈 설치

    https://github.com/0IhosuI0/GAN_Detector/releases/download/v{version}/GAN_Detector_v{version}.tar.xz

### 2. 압축 해제

    tar -xzf GAN_Detctor_v{version}.tar.xz

### 3. 실행

## 윈도우(Windows)

    .\start.bat

## Linux / MacOS

    chmod +x ./start.sh
    ./start.sh

자동으로 웹이 실행되지 않는 경우 http://localhost:5050 접속

모델이 켜지는데 30초 소요 될 수 있음
